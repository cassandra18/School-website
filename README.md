# School Website

A modern, responsive school website built with React, TypeScript, and Vite.

## 🌐 Live Demo
Visit the website: [https://school-website-five-delta.vercel.app](https://school-website-five-delta.vercel.app)

## 📋 Overview

This is a comprehensive school website project designed to provide information and resources for students, parents, and staff. The site features a modern, responsive design and intuitive user interface.

## ✨ Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI Components**: Built with React 19 and styled with Tailwind CSS
- **Smooth Animations**: Enhanced user experience with Framer Motion
- **Interactive Elements**: Carousel components, parallax effects, and smooth transitions
- **Data Visualization**: Charts and statistics using Recharts
- **Navigation**: Multi-page routing with React Router
- **Icon Library**: Comprehensive icon support with Lucide React and React Icons

## 🛠 Tech Stack

### Frontend Technologies
- **React 19**: Latest React framework for building interactive UIs
- **TypeScript**: Type-safe JavaScript for better code quality
- **Vite**: Fast build tool and development server
- **Tailwind CSS 4**: Utility-first CSS framework
- **React Router 7**: Client-side routing
- **Framer Motion**: Animation library for smooth transitions
- **Recharts**: Chart and data visualization library
- **Swiper & Slick**: Carousel and slider components

### Development Tools
- **ESLint**: Code quality and consistency
- **Autoprefixer**: CSS vendor prefixing
- **PostCSS**: CSS transformation

## 📦 Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/cassandra18/School-website.git
   cd School-website
   ```

2. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

## 🚀 Development

To start the development server:
```bash
npm run dev
```

The application will open at `http://localhost:5173` (default Vite port).

## 📦 Build

To build the project for production:
```bash
npm run build
```

The optimized build files will be created in the `dist` folder.

## 👀 Preview

To preview the production build locally:
```bash
npm run preview
```

## 🧹 Linting

To check code quality with ESLint:
```bash
npm run lint
```

## 🎨 Customization

- **Styling**: Modify Tailwind CSS configuration in `frontend/tailwind.config.js`
- **Build Settings**: Update Vite configuration in `frontend/vite.config.ts`
- **ESLint Rules**: Adjust ESLint rules in `eslint.config.js`

## 📝 Dependencies Overview

### Main Dependencies
- `react` & `react-dom`: Core React library

- `framer-motion`: Animations

- `vite`: Build tool
- `tailwindcss`: CSS framework
- `eslint`: Code linting

## 🚀 Deployment

The website is currently deployed on Vercel. To deploy your own version:

1. Push your code to GitHub
2. Connect your GitHub repository to Vercel
3. Vercel will automatically build and deploy on each push to the main branch
